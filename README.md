# TradingAgent AI

TradingAgent AI is an AI-powered trading research dashboard concept built with Next.js, TypeScript, Tailwind CSS and Recharts.

The project presents a modern financial dashboard interface for reviewing market signals, watchlists, risk exposure, portfolio performance and AI-assisted market insights.

> Disclaimer: This project is an educational frontend portfolio concept. It is not financial advice and does not execute real trades.

## About the Project

TradingAgent AI is designed as a fictional AI trading research dashboard.

The goal of the project is to demonstrate modern frontend development skills, dashboard UI design, responsive layouts, chart integration and clean component architecture.

The dashboard includes:

- AI trading research dashboard
- Sidebar navigation layout
- Market overview cards
- Watchlist section
- AI signal cards
- Risk exposure panel
- Recharts performance chart
- AI market assistant panel
- Dark financial dashboard UI
- Responsive mobile layout
- Vercel deployment

 Key Features
Dashboard-First Layout

Unlike a standard landing page, TradingAgent AI uses a dashboard-first layout with sidebar navigation, market overview cards and data-focused sections.

Market Overview

The dashboard displays portfolio value, AI confidence, open signals and risk exposure in a clean financial UI.

Watchlist

The watchlist section shows mock market assets with prices, percentage changes and sentiment labels.

AI Signals

TradingAgent AI includes mock AI-generated research signals with confidence levels, summaries and risk ratings.

Risk Panel

The risk panel presents a portfolio risk score, exposure indicators and a clear educational disclaimer.

Performance Chart

The project uses Recharts to display a responsive weekly momentum chart with custom tooltip styling and gradient area fill.

AI Market Assistant

The assistant panel simulates how an AI trading research assistant could summarize market conditions and compare signals.
